# 📊 Sales Performance Dashboard – Power BI

## 📌 Project Overview

This project presents a **Sales Performance Dashboard** built using **Power BI** to analyze company sales across multiple dimensions including products, categories, countries, and salespersons.

The dashboard provides insights into **revenue, profit, order volume, and product performance** to support data-driven decision making.

---

## 📊 Dashboard Features

### Sales Overview

* Revenue analysis by **quarter**
* Revenue distribution by **product category**
* Profit analysis by **country**

### Product Performance

* **Top selling products**
* **Lowest performing products**
* Profit and revenue analysis by product

### Salesperson Performance

* Sales revenue by **country and salesperson**
* Total sold items per salesperson
* Profit generated by each sales representative
